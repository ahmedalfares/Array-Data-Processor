# Array-Data-Processor



A modular **C-based** utility designed to process numerical arrays and extract fundamental statistics using functions.

## Features
* **Minimum Element:** Finds the smallest value in the array.
* **Maximum Element:** Finds the largest value in the array.
* **Sum:** Calculates the total sum of all elements.
* **Average:** Computes the arithmetic mean of the dataset.

## How to Run
1. **Compile the source code:**
   ```bash
   gcc main.c -o ArrayProcessor
2. Execute the program

Example output:
Enter number of elements: 5
Enter 5 integers: 10 20 30 40 50

--- Statistics ---
Minimum: 10
Maximum: 50
Sum: 150
Average: 30.0
